# Investigación de Ciberseguridad: futbollibretv.su en Dispositivos Móviles

## Resumen
Este proyecto documenta las vulnerabilidades, riesgos de privacidad y exfiltración de datos detectados al interactuar con el sitio futbollibretv.su utilizando un dispositivo Android.

## Metodología
- **Interceptación:** Burp Suite Community Edition.
- **Depuración:** Chrome DevTools (Remote Debugging).
- **Análisis de Reputación:** VirusTotal y AbuseIPDB.

## Estructura del Repositorio
- `/evidencias`: Capturas de pantalla y logs técnicos.
- `INFORME_TECNICO.md`: Análisis detallado de los hallazgos.
